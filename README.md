# Lazy Dashboard

A modern full-stack productivity dashboard with Kanban functionality built using React, Tailwind CSS, and Supabase.

## Features

- Authentication
- Kanban Board
- Drag & Drop
- Pomodoro Timer
- Productivity Analytics
- Responsive Dashboard UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- Supabase
- dnd-kit
